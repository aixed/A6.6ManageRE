# A++ 6.6 运维管理平台 (后端)

运行方式：

1.下载安装python 3.10

https://www.python.org/downloads/release/python-3100/



2.下载安装node-v16.13.1-x64

https://nodejs.org/zh-cn/



3.安装依赖包，控制台分别执行 

pip install cx-Oracle

pip install numpy

pip install flask

pip install flask-cors

pip install flask-httpauth

pip install flask-sqlalchemy



4运行

python.exe app.py
或
0.启动.bat
